Admin IP - Simply displays your IP Address (and hostname) on the WordPress dashboard admin panel.

== Description ==

Displays your IP address in your WordPress dashboard admin panel.
To avoid using external web site, there is a quick solution : Admin IP plugin.

This plugin simply display IP Address on the WordPress dashboard admin panel.

Very simple to install, simple to use.
Many translations.

== Installation ==

1. Upload `admin-ip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Is Admin IP free of charge ? =

Yes, this is a free WordPress plugin.

= How to use this plugin properly ? =

Install it. Activate it.
Then look for the admin address IP in the admin dashboard.
That's all.

= Can I show admin IP Address in a sidebar widget ? =

Absolutly not, nobody needs to know your admin IP Address.

= Where is the plugin settings ? =

There is no settings.

= Can I dispay Server IP Address or show User IP Address with this plugin ? =

Not possible. This plugin displays only server IP Address.
To display Server IP Address (hosting server IP), use this WordPress plugin :

http://wordpress.org/plugins/server-ip/

To show User IP Address (visitor IP), use this WordPress plugin :

http://wordpress.org/plugins/user-ip/
